# AI Coder - Termux

AI-powered coding assistant that runs in Termux.
Powered by Groq API (free & fast).

## Features
- Write code in any language
- Debug errors instantly
- Build apps step by step
- Review your files
- Supports Somali, English & Arabic
- Colorful terminal UI

## Setup

1. Install requirements
   pkg install python -y
   pip install requests

2. Get free API key
   https://console.groq.com

3. Add your key in ai.py

4. Run
   python ai.py
